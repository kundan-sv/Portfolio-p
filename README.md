# Personal Portfolio Website

A responsive personal portfolio built using React to showcase projects, skills, and hackathon experience.

## 🚀 Features
- Modular component-based architecture
- Sections: About, Skills, Projects, Hackathons, Contact
- Responsive design for all devices
- Clean and intuitive UI

## 🛠 Tech Stack
- React
- TypeScript
- Tailwind CSS

## 📂 Project Structure
- src/components → reusable UI components
- src/pages → main layout

## 📌 Description
This portfolio highlights my projects, technical skills, and experience through a structured and user-friendly interface.
